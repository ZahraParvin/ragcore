RAGCore

Multi-domain document intelligence powered by Claude AI. Upload documents, ask questions, run agentic data analysis.

Claude API RAG Pipeline Vector DB (Chroma) FastAPI Streamlit Docker Azure Ready

Features

📄

Multi-format ingestion

Upload PDFs, CSVs, or plain text. Automatic chunking and embedding into a persistent vector store.

💬

Conversational Q&A

Multi-turn chat over your documents with source citations. Claude retrieves relevant chunks before generating answers.

🔬

Agentic data analysis

Claude uses function calling to run statistical analysis, detect anomalies, and filter datasets — not just summarise them.

🗂️

Domain presets

Separate collections for industrial sensor data, financial reports, and general documents. Switch context instantly.
